Маршрутизатор платежа: тип операции, код и комплект документов

Мастер определяет правильный маршрут вашего платежа с лицевого счёта: прямой платёж контрагенту, возмещение собственных расходов или перечисление прибыли на расчётный счёт. Для каждого маршрута мастер выдаёт свой код направления расходования, обязательный комплект документов-оснований и шаблон назначения платежа: перепутанный маршрут почти всегда означает возврат распоряжения казначейством.

Маршрут: прямой платёж с лицевого счёта

Уровень риска: Низкий

Базовая операция: целевые средства уходят контрагенту напрямую. Нужны код направления по типу расхода и полный комплект оснований (договор, счёт, при поставке — закрывающие). Назначение — с ИГК и НДС.

Чек-лист действий

  1. Выберите код направления по типу расхода
  2. Соберите основания: договор + счёт (+акт/УПД)
  3. Соберите назначение: ИГК, предмет, реквизиты основания, НДС
  4. Сверьте остаток лимита и позицию Сведений
  5. Подайте РКП и отследите статус

Маршрут: возмещение расходов (коды 9XXX)

Уровень риска: Средний

Перевод «лицевой → расчётный» в компенсацию уже оплаченных затрат. Ключевое требование — доказать факт собственной оплаты: платёжка с расчётного счёта + документы поставки + привязка к контракту. Код 9XXX — по типу возмещаемого расхода.

Чек-лист действий

  1. Соберите доказательства оплаты с расчётного счёта
  2. Подберите код 9XXX по типу расхода
  3. Подтвердите связь расхода с контрактом
  4. Оформите назначение по шаблону возмещения
  5. Подайте РКП с полным пакетом

Маршрут: перечисление прибыли

Уровень риска: Средний

Вывод финансового результата возможен после приёмки: этап закрыт актом, прибыль подтверждена структурой цены. Пакет: акт, расчёт прибыли, корректный код операции. Попытка вывести до приёмки — гарантированный возврат.

Чек-лист действий

  1. Убедитесь: этап принят, акт подписан
  2. Рассчитайте прибыль от структуры цены контракта
  3. Подберите код операции вывода прибыли
  4. Приложите акт и расчёт к распоряжению

Маршрут: выплата зарплаты с лицевого счёта

Уровень риска: Средний

Зарплата персонала, занятого на контракте, — разрешённый расход: платится с ЛС по своему коду с подтверждением (ведомость, расчёт). Альтернатива — платить с расчётного и возмещать. Для регулярных выплат проверьте применимость упрощённого порядка (ПП № 2053).

Чек-лист действий

  1. Подтвердите занятость персонала на контракте
  2. Подготовьте ведомость/расчёт как основание
  3. Выберите код: прямая выплата или возмещение
  4. Проверьте применимость упрощённого перечисления
